Este projeto é uma aplicação desenvolvida com Spring Boot 3 e JPA para gerenciar pedidos de usuários, abrangendo o registro de novos pedidos, atualização do status de pagamento
 e o gerenciamento de produtos e suas categorias. 

Tecnologias Utilizadas:

- Spring Boot 3;
- JPA;
- Apache Tomcat;
- H2 Database (banco de dados em memória para testes);
- Maven (gerenciador de dependências);
- Postman (testes de API REST).

Principais Funcionalidades:

- Cadastro de usuários e pedidos;
- Registro de informações detalhadas dos pedidos (como quantidade de itens e valores);
- Atualização do status de pagamento dos pedidos;
- Gerenciamento de produtos e suas categorias;
- API RESTful para interação com a aplicação.

===================================================================================================================

This project is an application developed with Spring Boot 3 and JPA to manage user orders, including registering new orders, updating payment statuses, and managing products 
and product categories.

Technologies Used:

- Spring Boot 3;
- JPA;
- Apache Tomcat;
- H2 Database (in-memory database for testing);
- Maven (dependency management);
- Postman (REST API testing).

Key Features:

- User and order registration;
- Detailed order information (such as items and values);
- Payment status updates for orders;
- Product and category management;
- RESTful API for application interaction.
